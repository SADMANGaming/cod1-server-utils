# [CoD](https://en.wikipedia.org/wiki/Call_of_Duty_(video_game)) SERVER UTILS


### FEATURES:
- OneShot 
- Anti FastFire
- Mapvote
- Hud Text
- Chat Commands
- Only one ammo on pistol and grenade
- Remove placed items
- Ping checker
- TDM, BEL all weapons from both team and others are bolt weapons only
- Alive players HUD
- Score HUD
- 1.5 Headicons
